# Projekta nosaukums

## Apraksts
- Šī spēle satur aritmētiskas pamatproblēmas
- Katra pareiza atbilde tiks pielikta pie 1 Punkta
- Pilni jautājumi būs
- Uzvarai, jums ir vajadzīgi 3 vai vairāk pareizu atbildes
## Galvenās funkcijas
	- Ievadīt tekstu
	- Apstrādāt datus
	- dod vajadzīgu informaciju klientiem
  - izvadīt datus

	- SFML library
  - Python
