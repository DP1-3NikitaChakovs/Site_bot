## Apraksts
- Tas ir saita bots, kurš pēc jūsu jautajumiem, var jums palīdzet vai parvēst klientu uz specialistu kurš var palīdzet ar klienta jautajumu.

	- Apstrādāt datus
	- dod vajadzīgu informaciju klientiem
  - izvadīt datus
